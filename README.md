# Social Network Ads Predictor

This project trains a machine learning model to predict whether a user will purchase a product based on gender, age, and estimated salary.

## Files

- [assignment_model.py](assignment_model.py): trains the model and saves it to `bestmodel.pkl`
- [app.py](app.py): launches the Gradio demo UI
- [social_network_ads_dataset.csv](social_network_ads_dataset.csv): dataset used for training
- [requirements.txt](requirements.txt): Python dependencies

## Run

1. Create and activate the virtual environment.
2. Install dependencies with `pip install -r requirements.txt`.
3. Train the model with `python assignment_model.py`.
4. Start the app with `python app.py`.
